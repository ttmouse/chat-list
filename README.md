# 话术助手 - 浏览器插件

一个功能强大的浏览器插件，帮助您在任何网页上快速使用和管理常用话术。

## 功能特点

### 🎯 核心功能
- **悬浮话术列表**：在网页右侧显示悬浮的话术列表
- **一键填充**：点击话术内容即可自动填充到当前输入框
- **话术管理**：支持添加、编辑、删除话术
- **分组管理**：可以将话术按类别分组，便于管理
- **智能识别**：自动识别页面中的输入框并填充内容

### 🛠️ 管理功能
- **分组标签**：支持为话术创建不同颜色的分组
- **快速筛选**：可按分组筛选显示话术
- **数据导入导出**：支持备份和恢复话术数据
- **设置选项**：可自定义显示方式和行为

### 🎨 界面特色
- **现代化设计**：美观的渐变色界面
- **响应式布局**：适配不同屏幕尺寸
- **流畅动画**：丰富的交互动画效果
- **可拖拽定位**：可以拖拽到任意位置

## 安装方法

### 开发者模式安装
1. 打开 Chrome 浏览器
2. 访问 `chrome://extensions/`
3. 开启右上角的「开发者模式」
4. 点击「加载已解压的扩展程序」
5. 选择本项目文件夹
6. 插件安装完成

## 使用指南

### 基本使用
1. **查看话术列表**：安装插件后，在任意网页右侧会显示悬浮的话术助手
2. **使用话术**：点击任意话术内容，会自动填充到当前焦点的输入框
3. **切换分组**：点击分组标签可以筛选不同类别的话术
4. **收起展开**：点击标题栏的📋按钮可以收起或展开话术列表

### 话术管理
1. **添加话术**：
   - 点击「+ 添加话术」按钮
   - 或点击设置按钮⚙️进入管理面板
   - 填写话术标题、选择分组、输入内容
   - 点击保存

2. **编辑话术**：
   - 鼠标悬停在话术上，点击✏️编辑按钮
   - 修改内容后点击保存

3. **删除话术**：
   - 鼠标悬停在话术上，点击🗑️删除按钮
   - 确认删除

### 分组管理
1. **添加分组**：
   - 进入管理面板
   - 在分组管理区域点击「+ 添加分组」
   - 输入分组名称

2. **编辑分组**：
   - 在分组列表中点击「编辑」按钮
   - 修改分组名称

3. **删除分组**：
   - 在分组列表中点击「删除」按钮
   - 该分组下的话术会移到未分组

### 设置选项
点击插件图标打开设置面板：
- **自动显示话术助手**：页面加载时自动显示
- **允许拖拽位置**：可以拖拽话术助手到任意位置
- **记住上次位置**：记住拖拽后的位置

### 数据管理
1. **导出数据**：
   - 点击「导入/导出话术」
   - 选择「导出话术数据」
   - 下载JSON格式的备份文件

2. **导入数据**：
   - 点击「导入/导出话术」
   - 选择「导入话术数据」
   - 选择之前导出的JSON文件
